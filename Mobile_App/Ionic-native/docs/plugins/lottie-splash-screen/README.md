# Lottie Splash Screen

```
$ ''
$ npm install @awesome-cordova-plugins/lottie-splash-screen
```

## [Usage Documentation](https://danielsogl.gitbook.io/awesome-cordova-plugins/plugins/lottie-splash-screen/)
